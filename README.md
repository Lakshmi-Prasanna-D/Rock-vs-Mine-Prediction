# Rock vs Mine Prediction (Machine Learning)

This project builds a binary classification model using Logistic Regression to classify sonar signals as Rock or Mine.

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn

## Model Details
- StandardScaler applied
- Stratified train-test split
- Logistic Regression with L2 regularization
- Achieved 84% test accuracy
